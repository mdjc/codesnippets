Keep your own library of code snippets

Keep organized those pieces of code you use more often or will use someday. Quick access to your library.

What is a code snippet?

is a reusable code that you can insert in your code or can use as concise example.

What to save?

snippets can be simple or more complex. Save a couple lines you cannot remember easily, a method, an entire class, a data structure, any tricky code...

Can I view others' snippets?

definitely yes! All snippets are public. The idea is to share yours as well as benefit from other users' libraries. all snippets here.
